# Projet-Fin-Session-A25
Projet de fin de session A25 Electronique programmable

Ce repo git a été créé dans le câdre du projet de fin de session d'automne 2025 du programme éléctronique programmable du Cégep Limoilou.
Cette branche est pour le code du Raspberry pi en C. Ce code sert de pont entre l'interface usager et le reste du systeme.

This git repo was created in the context of the end of autumn 2025 cycle project for the programmable electronics programm at the Limoilou College.
This branch is for the raspberry pi's code in C. This code is used as a bridge between the user interface and the rest of the system.
